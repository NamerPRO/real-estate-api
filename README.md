# Real Estate API: Серия лабораторных работ

> **Дисциплина:** Программная инженерия  
> **Автор:** Авдиянц Петр | Группа: **М8О-106СВ-25**

Этот репозиторий содержит серию лабораторных работ (**ЛР-2 - ЛР-5**), посвященных проектированию, разработке и оптимизации высоконагруженного REST API сервиса для управления недвижимостью. Проект реализован на **C++20** с использованием фреймворка **Userver**.

## Структура репозитория

Для удобства навигации и проверки исторических версий код организован двумя способами:

1.  **По папкам:** Каждая лабораторная работа выделена в отдельную директорию в ветке main.
    *   [`real-estate-api-lr-2/`](https://github.com/NamerPRO/real-estate-api/tree/main/real-estate-api-lr-2), [`real-estate-api-lr-3/`](https://github.com/NamerPRO/real-estate-api/tree/main/real-estate-api-lr-3), [`real-estate-api-lr-4/`](https://github.com/NamerPRO/real-estate-api/tree/main/real-estate-api-lr-4), [`real-estate-api-lr-5/`](https://github.com/NamerPRO/real-estate-api/tree/main/real-estate-api-lr-5)
2.  **По веткам:** Полные версии проектов доступны в соответствующих изолированных ветках:
    *   [`lr-2`](https://github.com/NamerPRO/real-estate-api/tree/lr-2), [`lr-3`](https://github.com/NamerPRO/real-estate-api/tree/lr-3), [`lr-4`](https://github.com/NamerPRO/real-estate-api/tree/lr-4), [`lr-5`](https://github.com/NamerPRO/real-estate-api/tree/lr-5)

В каждой из лабораторных работ подробно описана инструкция по ее запуску и проверке.
